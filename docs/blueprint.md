# **App Name**: DataSight - VitaLake

## Core Features:

- Database Browser: Display database connections and navigate to Schema Browser.
- Schema Browser: Show database schemas and tables.
- Table/Data View: Render a table view of data with column metadata and a button to query the table in SQL Studio.
- SQL Studio: Two-column layout with Schema Explorer and SQL Editor. Run queries and display results in a table.
- Chart Builder: Create charts from query results with live preview and save functionality.
- Dashboard Builder & Viewer: Create and view dashboards with drag-and-drop grid layout for arranging charts and tables.
- Global Navigation: Sidebar/Topbar navigation with breadcrumbs and responsive layout.

## Style Guidelines:

- Background: #0F1113 (base background)
- Surface / Card background: #1A1C1E
- Primary text: #FFFFFF (faint white)
- Secondary / muted text: #A3A6A8
- Accent / Titanium-Blue brand color: #5B798E
- Accent-light (hover/active): #708CA4
- Borders / dividers: #2C2E30
- Error / alert color: #E06C75
- Body / UI Font: Inter (sans-serif)
- Code / SQL editor: Source Code Pro (monospace)
- Minimal icons from Heroicons or lucide-react
- Responsive layout using Tailwind CSS grid system
- Smooth transitions and hover/active states styled with accent & accent-light
